Desestructuración : Consiste en separar una estructura ( en JS se hace con un array o un objeto) const matriz = [5, 6]; const [first, last] = matriz; //first = 5, last = 2 const matriz = [5, 4, 3, 2]; const [first, second] = matriz; //first = 5, second = 4, rest = discard-descartadas const matriz = [5, 4, 3, 2]; const [first, third] = matriz; //first = 5, third = 3, rest = discard const matriz = [4]; const [first, second] = matriz; //first = 4, second = undefined En la parte de la derecha los [] indican que se trata de un array, pero en la parte de la izquierda se utilizan para indicar que se hace una desestructuración.
